## **The Budget Info App - Budget_Info:**  
  
### **Overview & First Steps**  
The Budget Info project offers a simple and user-friendly way to manage the remaining monthly budget.  

- **Short Description:**  
....  
  
- **Multilingualism:**
German and English  
  
- **Supported Platforms:**  
Android and Windows  
  
- **Required Permissions:**  
The app only saves its settings to its own document directory and does not require any other permissions. If the user selects Export or Import from the main menu, files must be saved, shared, and read outside the app to save and synchronize the settings.  
  
- **Quick Start Guide for Windows:**  
Copy the ZIP file to a new directory of your choice (e.g., C:\Tools\BudgetInfo) and extract it, for example, using File Explorer. Right-click on the file "budget_info.exe" and select "Pin to Start" so you can easily access it from the Start menu. For updates, simply extract the new ZIP file back into this directory and overwrite the old files.  
  
- **Quick Start Guide for Android:**  
As long as the app is not yet visible in the Google Play Store, save the APK file to your Android tablet or phone and install it using a file manager that has permission (e.g., Google's "Files" app or your browser).

### **Details**  
  
- **Timetable Management / Export/Import**  
The export/import function allows you to create a complete backup of your data and settings at any time. This should be done regularly after making changes. The resulting ZIP file, containing the date, can be saved by the user and imported again at any time. This allows you to manage years, try out different things and so on. Furthermore, data can be easily exchanged between the Windows and Android apps. The app data is identical, but may change from version to version. The same version should always be used on all devices. The version can be found at the top of the home screen menu.  
To delete all files (restore to factory settings), you can do so via the Android built-in "App Info" and "Clear Storage," or using Windows File Explorer in the "Documents" folder under "BudgetInfo". Of course, close the app beforehand.  
  
- **Work Plan / Manage Tasks/Steps**  
Essentially the list is a monthly plan for a year. The lists can, of course, also be used for other purposes. Because the text in the list and the filename used to store and reload entries in the lists can be freely chosen, there are no limits to what can be entered in these lists. For example, there could be a task list. Of course, by repeatedly changing the filename of the suggestion list, any tasks from multiple lists can be added to the task list on the left side, and so on.  
  
- **Additional buttons in the work plan view**  
The "+ Main" button adds a main line to the list, which can then be moved using the up/down arrow buttons. With a doubleclick on a line the text is editable.   
If a task is selected, another step can be added below the selected task using the "+ Sub" button and moved using the corresponding up/down arrow buttons. With a doubleclick on a line the text is editable.  
The two buttons "<= Main" and "<= Sub" allow you to copy entire tasks or individual work steps from the right-hand list to the left-hand list. To do this, a task or work step must be selected on the right, and in the case of a work step, a task must also be selected on the left to insert the step.  
The red minus sign in a circle deletes an entire block or a line from a list.  
All changes must be saved via the "Save" button.  
   
- **Using Status Management in the Task List**  
The list features a colored status indicator, at the beginning of the step line, to represent the current status of the step. Currently, the colored circles are yellow for planned, and green finished. Clicking the colored circle changes the status in the sequence yellow, green and back to yellow. This makes it easy to see which lines have already been completed and which are still pending for each step. Once all steps for a block are complete, the block is displayed collapsed when the window is opened, improving clarity.
